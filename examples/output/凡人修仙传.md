[动作] 韩立从山上推门走进前屋，肩上还挎着沾着泥土的藤筐。

### 场景 3: 离别出发

| 属性 | 值 |
|------|------|
| 地点 | 村口 |
| 时间 | 清晨 |
| 氛围 | 离别伤感、希望 |

**出场角色**: 韩立(强忍泪水，心中坚定)、韩父(反复嘱咐)、韩母(含泪叮嘱)、三叔(催促上路)

[动作] 清晨的村口，薄雾未散。一辆青布马车从土路远处驶来，渐渐停在大槐树下。三叔掀开车帘跳下地，圆脸上带着笑，身上缎子衣服在晨光中发亮。

[动作] 韩立背着一个小布包袱，站在父母中间。韩母眼圈发红，一手紧紧攥着儿子的衣角，嘴巴翕动却只说出一句：“到了那边，别忘了给我们捎信。”韩父沉默着，把旱烟杆从嘴里拿下来，拍了拍韩立的肩膀，喉咙里含糊地“嗯”了一声。

[动作] 韩立抬起头，嘴唇抿得紧紧的，眼眶发酸却拼命忍住不让泪水掉下来。他用力点头，朝父母弯腰鞠了一躬，然后转身快步走向马车。

[动作] 三叔伸手搀了韩立一把，帮他登上马车，随后自己也跳上车辕。他回头朝韩父韩母拱了拱手：“大哥大嫂放心，我会照看好立儿。”说罢一抖缰绳，马车缓缓启动。

[动作] 车轮碾过黄土路，扬起细尘。韩立从车帘缝里最后看了一眼村口——父母的身影越来越小，韩母抬起袖子擦眼睛，韩父依然站在原地佝偻着腰。韩立狠狠闭上眼，把眼泪逼回眼眶里，攥紧包袱带子。

### 场景 4: 春香酒楼后院

| 属性 | 值 |
|------|------|
| 地点 | 春香酒楼后院厢房 |
| 时间 | 傍晚至夜 |
| 氛围 | 放松、新奇、舒适 |

**出场角色**: 韩立(休息、听故事)、三叔(照料、吹嘘江湖经历)

[动作] 傍晚，韩立跟随三叔穿过春香酒楼喧闹的大堂，来到幽静的后院。三叔推开一间厢房的门，屋内陈设简单，床铺整洁。

三叔（亲切）: "立儿，先好好休息，赶路辛苦了。明儿个我让伙计给你送些吃的来。"

[动作] 韩立点头，在屋内坐下。窗外传来街市的嘈杂声，他望着陌生的环境，心中既好奇又有些忐忑。

[动作] 次日傍晚，韩立与三叔在院中石凳上对坐。三叔抽着旱烟，开始讲述年轻时的江湖见闻——与人比武、暗器奇毒、深山寻宝。韩立听得入神，不时问几句。

> [旁白] 就这样，韩立在春香酒楼后院度过了两天平静的日子，除了听三叔讲故事，便是独自休息养神。他隐隐感到，自己平凡的生活即将被打破。

[动作] 韩立稳住身形，回头看了一眼深不见底的山崖，深吸一口气，对张均点了点头，重新抓住岩石继续向上攀爬。

| 时间 | 接近正午 |

**出场角色**: 韩立(虚脱、沮丧)、张均(抱起韩立攀升)、舞岩(在崖顶挑衅嘲笑)

[动作] 韩立气喘吁吁地跑到悬崖底，抬头望向崖顶。舞岩的身影已经站在上方，双手抱胸，俯视着他。

舞岩（嘲讽）: "就这点本事，也敢来闯炼骨崖？我看你还是滚回去放牛吧！"

[动作] 韩立咬了咬牙，伸手抓住身旁垂下的麻绳，双臂用力向上拉，身体刚离地半尺就滑落下来。他连续试了三次，每次都是攀到一半就因力竭而坠落，手掌被麻绳磨得通红。

[动作] 张均从后面快步上前，一言不发地弯腰，双手抄起韩立的腋下，将他整个人抱了起来。

[动作] 张均抱着韩立，走到崖壁前，脚下用力一蹬，单手抓住麻绳，几个纵跃间便轻松攀上崖顶，稳稳落地后放下韩立。

### 场景 11: 结果宣布

| 属性 | 值 |
|------|------|

**出场角色**: 韩立(记名弟子，略带失落)、张铁(同样被选为记名弟子)、舞岩(以胜利者姿态)、岳堂主(宣布结果)、王护法(在旁)、蓝袍老者(满意地看着舞岩)

### 场景 12: 偶遇墨大夫

| 属性 | 值 |
|------|------|
| 地点 | 下山小路（途经树林） |
| 时间 | 午后 |
| 氛围 | 意外、恭敬、顺从 |

**出场角色**: 韩立(跟随，茫然)、张铁(同样被选中)、张均(恭敬行礼)、吴铭瑞(拍马屁，虽瘦长脸)、墨大夫(咳嗽，随意指定)

[动作] 午后的树林中，张均和吴铭瑞带着韩立、张铁等六名孩童沿着下山小路前行。阳光透过树叶洒下斑驳光影，孩子们神情紧张。

[动作] 前方树影摇晃，一个高瘦、面皮焦黄、白发的老者拄着拐杖缓步走出，咳嗽声在林中回响。正是墨大夫。

吴铭瑞（殷勤）: "咦？这不是墨师伯吗？您老人家怎么在这儿？"

张均（恭敬）: "墨前辈安好。"

墨大夫（沙哑，漫不经心）: "嗯，你们这是带着新弟子去百锻堂？"

吴铭瑞（谄媚）: "正是，墨师伯。这些都是今日入门的记名弟子，正要送过去让岳堂主过目。"

[动作] 墨大夫缓缓走近，目光在六名孩童脸上逐一掠过，最后停在韩立和张铁身上。

墨大夫（不容置疑）: "这两个孩子，我要了。一个给我当炼药童子，一个给我采药。"

韩立（内心独白·）: "炼药童子？采药？这位老大夫看起来不像坏人……但为何偏偏选我们两个？"

张均（平静）: "墨前辈选中他们，是他们的福气。晚辈这就带其余四人下山。"

[动作] 吴铭瑞连连点头，不敢有异议，赶紧招呼其他四个孩子继续前行。韩立和张铁站在原地，望着师兄弟们的背影消失在林间小径。

墨大夫（低沉，略带疲惫）: "跟我来。"

### 场景 13: 神手谷安顿

| 属性 | 值 |
|------|------|
| 地点 | 神手谷 |
| 时间 | 黄昏 |
| 氛围 | 宁静、药香、疲惫 |

**出场角色**: 韩立(疲惫入睡)、张铁(同样被安置)、墨大夫(咳嗽着离开)

[动作] 黄昏时分，墨大夫领着韩立和张铁走进神手谷。山谷中雾气缭绕，几间木屋稀疏地散布在绿树之间。墨大夫指着一排房屋，边走边用沙哑的声音简单介绍布局。

[动作] 墨大夫推开一间厢房的门，示意韩立和张铁看里面的床铺和简单陈设。他咳嗽两声，交代了几句院落规矩，然后转身走出房门。

[动作] 墨大夫的身影消失在山谷小径的拐角处。韩立站在房门口，面色疲惫，眼皮沉重。

[动作] 韩立再也撑不住，转身跌跌撞撞走到床边，一头倒在粗糙的被褥上，几乎立刻沉沉睡去。张铁在隔壁房间也传来轻微的鼾声。
